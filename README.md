# 💕 Convite Especial

Uma página interativa e fofa para fazer um convite para um date! Com animações, botão que foge e tela de sucesso.

## 🎯 Como Acessar

Você pode acessar a aplicação em:

**[https://omurilomvp-cyber.github.io/Convite-especial/](https://omurilomvp-cyber.github.io/Convite-especial/)**

## 🎨 Recursos

- ✨ Página responsiva simulando um smartphone
- 🥺 Emoji animado com efeito de pulsação
- 🏃 Botão "NÃO" que foge do cursor
- 💕 Corações flutuando ao fundo
- 🎉 Tela de sucesso ao aceitar o convite

## 🚀 Tecnologias

- HTML5
- CSS3 (com animações e gradientes)
- JavaScript vanilla

## 📱 Como Usar

Simplesmente abra o link acima e interaja com a página! Tente clicar no botão "NÃO" 😊
